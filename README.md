# Customer Segmentation & Marketing Campaign Dashboard (Tableau)
Data Analytics & Business Intelligence

## Project Overview

This project analyzes customer demographics, spending behavior, and marketing campaign performance using an interactive Tableau dashboard.

The objective was to identify key customer segments, understand spending patterns, and evaluate how different groups respond to marketing campaigns.

## Tools Used

* Tableau (Data Visualization)
* Excel (Data Preparation)
* Marketing Campaign Dataset

## Dashboard Insights

### 1. Spending Patterns by Age and Marital Status

The analysis shows that higher income does not always translate to the highest spending. Certain age groups and marital status segments contribute more significantly to total spending.

### 2. Education Level and Spending

Customers with **Graduation-level education** showed the highest average spending, slightly exceeding those with a Master's degree. Lower education groups generally fall below the average spending benchmark.

### 3. Campaign Response by Education Level

Customers with **PhD-level education showed the highest campaign response rate**, indicating stronger engagement with marketing activities. Customers with **Basic education demonstrated the lowest response rate**.

### 4. Campaign Response by Marital Status

Marketing response patterns vary across marital status groups, with some household structures responding more actively to campaigns than others.

## Key Business Value

The dashboard demonstrates how customer segmentation and behavioral analysis can help businesses:

* Identify high-value customer groups
* Improve marketing campaign targeting
* Understand spending patterns across demographic segments
* Support data-driven marketing strategies
